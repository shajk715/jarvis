# 🎨 Frontend Agent

## 역할 요약
자비스의 얼굴을 만드는 에이전트. 음성 AI인 만큼 화면은 심플하되, 자비스가 듣고/생각하고/말하는 상태를 파동 애니메이션으로 표현한다.

---

## 책임 범위
- `index.html` — 메인 화면 구조
- `style.css` — 파동 애니메이션 및 전체 스타일
- 모바일 반응형 레이아웃
- 상태별 UI 변화 (대기 / 듣는 중 / 생각 중 / 말하는 중)

---

## 디자인 컨셉

### 전체 분위기
- 다크 테마 (검정/진한 남색 배경)
- 중앙에 큰 파동(원형 웨이브) 애니메이션
- 미래적이고 고급스러운 느낌 (아이언맨 자비스 UI 참고)
- 텍스트 최소화 — 음성이 메인이므로 화면은 상태 표시만

### 상태별 UI

| 상태 | 파동 | 색상 | 텍스트 |
|------|------|------|--------|
| 대기 중 | 천천히 맥박 | 흰색/회색 | "자비스라고 불러주세요" |
| 듣는 중 | 빠르게 출렁 | 파란색 | "듣고 있습니다, 주인님" |
| 생각 중 | 회전하며 맥박 | 보라색 | "처리 중입니다..." |
| 말하는 중 | 소리에 맞춰 출렁 | 초록색 | (자비스 답변 텍스트 표시) |

---

## 구현 사항

### 파동 애니메이션
- CSS `@keyframes` 또는 Canvas API 사용
- 여러 겹의 원이 퍼져나가는 ripple 효과
- 상태에 따라 애니메이션 속도/색상 변화

### 레이아웃
```
┌─────────────────────┐
│                     │
│                     │
│      ◉ 파동 ◉       │  ← 화면 중앙 70% 차지
│                     │
│                     │
│  "듣고 있습니다"     │  ← 하단 상태 텍스트
│                     │
└─────────────────────┘
```

### 모바일 최적화
- viewport meta 태그 필수
- 터치 이벤트 지원
- 화면 꺼짐 방지 (Wake Lock API 사용)
- 전체화면 지원

---

## 외부 연결 (다른 에이전트와 연동)
Voice Agent, AI Agent에서 아래 함수를 호출하여 상태 변경:

```javascript
// 상태 변경 함수 (Frontend에서 export)
setJarvisState('idle')       // 대기
setJarvisState('listening')  // 듣는 중
setJarvisState('thinking')   // 생각 중
setJarvisState('speaking')   // 말하는 중
```
